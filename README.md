# Retail Customer Basket Analysis

**Tools:** Python · pandas · matplotlib  
**Domain:** E-Commerce / Retail  
**Dataset:** 15,000 basket records + 20,000 customer profiles

## Project Overview

Analysed retail customer purchase behaviour using two datasets — basket transactions and customer profiles — to uncover patterns in when customers buy, how much they buy, and how demographics affect purchase behaviour.

## Key Business Insights

1. **Purchase peaked on 27 May 2019** with 3,478 items sold in a single day — likely a promotion. Recommend replicating this offer in future cycles.

2. **Female customers buy 13% more per visit** — average basket size 2.38 vs 2.11 for male customers. Bundle promotions could increase male basket size.

3. **New customers buy slightly more per visit** — 0 to 30 day customers averaged 2.24 items vs 2.11 for 90+ day customers.

4. **37.3% of age records are invalid** — values like -34 and 2022 exist in the data. Data validation rules must be added at the point of collection.
